Création d'un site Wordpress

Demande client : Rajeunir un blog WordPress existant

Détermination des objectifs :

  • Remettre le design au goût du jour
  • Permettre au client de pouvoir facilement publier
  • Rajouter de l'intéractivité : par exemple formulaires
  • Site responsive

Choix effectués :

  • Pour simplifier l'utilisation du site de ma cliente, et choix a été de conserver Wordpress
  • Pour pouvoir obtenir le résultat visuel escompté sans être bridé par des thèmes existants, mon choix s'est porté sur un Starter Theme basic Underscores
  • Utilisation du framework Boostrap pour le CSS3
  • Installation de divers plugins existants, tels que Formidable(formulaire), Yoast SEO(pour le référencement) ...
